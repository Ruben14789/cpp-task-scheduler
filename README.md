# C++ Multithreaded Task Scheduler

A multithreaded task scheduler implemented in C++ using a thread pool, condition variables. This project demonstrates asynchronous execution and performance improvements using parallel processing.

##  Features

- Fixed-size thread pool
- Thread-safe task queue
- Asynchronous task execution with `std::future`
- Graceful shutdown of worker threads
- Demonstrates real parallel speedup
